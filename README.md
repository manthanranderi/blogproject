# 🚀 Admin Panel Management System

A professional **Admin Panel Management System** built using **Node.js, Express.js, MongoDB, Mongoose, EJS, and Multer**. This project provides a simple dashboard where an administrator can manage records with complete CRUD (Create, Read, Update, Delete) functionality.

# 📌 Project Overview

The Admin Panel Management System is designed to manage administrator information efficiently. It provides a clean interface for adding, viewing, updating, and deleting admin records while storing data securely in MongoDB.

This project follows the MVC (Model-View-Controller) architecture for better code organization and maintainability.

---

# ✨ Features

* Admin Dashboard
* Add New Admin
* View Admin Details
* Update Existing Admin
* Delete Admin
* Upload Profile Image
* Form Validation

# ⚙ Installation

## Clone the repository

```bash
git clone <repository-url>
```

## Go to project folder

```bash
cd Admin-panel
```

## Install dependencies

```bash
npm install
```

## Start the project

```bash
npm start
```

---

# 🌐 Server

The application starts using:

```bash
npm start
```

which runs

```bash
nodemon app.js
```
